# Nigerian Fuel Subsidy Economic Impact Analysis

An economic data science project analysing the impact of Nigeria's May 2023 fuel subsidy removal on key macroeconomic indicators — inflation, food prices, transport costs, and exchange rates.

## Project Structure
```
fuel-subsidy-impact/
├── data/
│   ├── raw/          # Raw downloaded datasets (gitignored)
│   └── processed/    # Cleaned and merged datasets (gitignored)
├── notebooks/        # Jupyter notebooks for exploration and storytelling
├── src/              # Reusable Python modules
├── tests/            # Pytest unit tests
└── outputs/          # Charts and exports (gitignored)
```

## Setup
```bash
conda activate fuel-subsidy
jupyter notebook
```

## Data Sources
- NBS (National Bureau of Statistics Nigeria)
- CBN (Central Bank of Nigeria)
- World Bank Open Data
- NNPC/PPPRA
